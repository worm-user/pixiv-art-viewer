# Pixiv Reference Manager (仮)

絵描きの資料収集・閲覧を強力にサポートするデスクトップアプリケーションです。
直感的な操作で画像を拡大縮小しながら、高度なスポイト機能による色抽出などが行なえます。

## 主な機能

- **動的ギャラリー**
  - 画面サイズに合わせて画像枠が自動調整される見やすいグリッド表示
  - サイドバーへのホバーで展開するUIにより、画面を広々と使用可能
- **高性能イメージビューア**
  - **拡大・移動**: スクロールでズーム、右クリックドラッグで画像のパン（移動）が可能
  - **高度なスポイトツール**: 
    - マウスホバー位置を大きく拡大表示するルーペ機能
    - 左クリックで色を抽出し、カラーコードをクリップボードに自動コピー
    - 抽出エリアのサイズ調整機能（1x1, 3x3, 5x5, 9x9ピクセルから平均色を計算可能）
- **スマートカラーパレット (MRU)**
  - 自動抽出された主要色と、スポイトで取得した色をコンパクトに一覧表示
  - 最新の抽出色がパレットの先頭にストックされる先入れ先出し(FIFO)管理
  - お気に入りの色を消さずに残す「ピン留め」機能（右クリックでトグル）
- **ファイル連携機能**
  - 現在表示中の画像をエクスプローラーで即座に開く機能
  - 画像データ自体をクリップボードにコピーして他ツール（クリスタやPhotoshop等）にペーストする機能
- **ローカルAIによる自動タグ付け** (※開発中/一時無効化)
  - ローカルで動作するOllama (Qwen2-VL等) と連携した、画像の構図・スタイルの自動解析機能

## マウス操作一覧 (イメージビューア画面)

| 操作 | アクション |
| --- | --- |
| **マウスホイール** | 画像の拡大 / 縮小 |
| **左クリック** | マウスカーソル下の色を抽出・コピー（スポイト） |
| **右クリック + ドラッグ** | 画像の移動（パン） |
| **右ダブルクリック** | 画像の表示位置・倍率を初期状態にリセット |

*(カラーパレット上の操作)*
- **左クリック**: 選択した色をクリップボードにコピー
- **右クリック**: 色のピン留め / 解除のトグル

## 技術スタック

- **フロントエンド**: React, TypeScript, Vite
- **バックエンド/デスクトップ**: Electron, Node.js
- **主要ライブラリ**: 
  - `node-vibrant` (色抽出)
  - `lucide-react` (アイコン)
  - `ollama` (AIタグ付け連携)

## 開発・セットアップ

```bash
# 依存関係のインストール
npm install

# 開発用サーバーの起動
npm run dev

# アプリケーションのビルド
npm run build
```
